V4.65版本
增加虚拟声卡驱动的支持.(V1.0)
 使用方式：
    安装驱动目录(driver)下的audrv的虚拟声卡驱动，安装成功后控制面板的音频列表将多出一个ICC301的声卡设备，
    使用 ODT_AUDRV类型打开启用虚拟声卡的功能,如果没有打开，该虚拟声卡就不能被正常打开使用,如：mediaplay播放器选择该声卡时就会提示打开声卡失败
    演示方式：
	vc内部测试程序2.05版本在打开USB设备成功后就自动启动虚拟声卡功能，windows系统的播放器选择使用该虚拟声卡播放时语音就会达到该设备，选择界面上'打开喇叭'就可以听到。

 使用新增加的ICC301的API (QNV_Audrv)系列函数来对该虚拟声卡进行相关控制

如：
	if(QNV_Audrv(QNV_AUDRV_ISINSTALL,0,NULL,NULL,0) > 0)
	{	
		//虚拟声卡驱动已经安装
	}else
	{
		//系统未安装ICC301虚拟声卡驱动
	}
	if(QNV_OpenDevice(ODT_AUDRV,0,0) > 0)//启动虚拟声卡功能
	{
	QNV_Audrv(QNV_AUDRV_WAVEOUTID,0,NULL,NULL,0);//设置虚拟声卡的语音输出通道为0
	QNV_Audrv(QNV_AUDRV_WAVEINID,0,NULL,NULL,0);//设置虚拟声卡的语音输入通道为0
	}else
	{
	//启动ICC301虚拟声卡功能失败
	}


如果使用mediaplay演示播放步骤：
1.插入设备
2.安装驱动目录(driver)下的audrv的虚拟声卡驱动
3.打开控制面板,选择默认音频输出为ICC301
4.启动内部测试软件(qnviccubdemo.exe)
5.点打开设备
6.点打开喇叭
7.启动mediaplay播放语音，就可以从盒子喇叭可以听到


如果设备话机工作：
8.点'断开电话机'
9.拿起话机就自动切换到话机，可以从话机听到播放的语音,盒子喇叭就不出现声音。

x.启动QQ音频测试向导等其它网络电话的软件音频测试向导选择ICC301声卡也可以演示
